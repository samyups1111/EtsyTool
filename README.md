# EtsyTool

## Intro:

An app used to search for Etsy shops using the Etsy API. Users can click on the shops to see all the listings available. 

## Architecture:

* MVVM Architecture:
  * Model
  * View
  * ViewModel   
  
## Libraries

* Room Persistence Library
* Retrofit Library
* Moshi Library
* Coroutine
* Picasso
